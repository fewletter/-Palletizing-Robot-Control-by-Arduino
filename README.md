# -Palletizing-Robot-Control-by-Arduino
大三機械專題,使用碼垛型機器人來達成專題需求

控制原理:
1.	利用機動學算出末端(夾爪部分)的位置
2.	為了能讓馬達轉到我們所要的角度，我們設計一個矩形座標讓夾爪到某個定點後能傳輸兩個角度給兩個馬達。
